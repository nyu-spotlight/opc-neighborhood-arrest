*Bennett Allen, Brandi Moore, Victoria A Jent, William C Goedel, Khadija Israel, Alexandra B Collins, Brandon D L Marshall, Magdalena Cerda*

Script description:
- `OPC_Arrest_Primary_Analysis_Code.R` aggregates arrests to buffers, integrates ACS covariates, and runs all synthetic control models for the primary analysis.
